FixifiedTranslator

v1.3.0

Описание

FixifiedTranslator — это мощный мод для Minecraft Fabric, который автоматически переводит входящие сообщения чата на выбранный вами язык. Забудьте о языковых барьерах на международных серверах!

Возможности

  • ✅ Автоматический перевод всех входящих сообщений
  • ✅ Поддержка 15 языков
  • ✅ Синхронный перевод через Google Translate API
  • ✅ Без миксинов (совместим с Sodium)
  • ✅ Настройки через клавишу F9
  • ✅ Конфиг сохраняется в config/fixifiedtranslator.json
  • ✅ Работает на Fabric 1.21.11

Установка

  1. Установите Fabric Loader
  2. Скачайте мод и поместите .jar файл в папку mods
  3. Запустите Minecraft с Fabric профилем
  4. Нажмите F9 в игре для настройки языка

Системные требования

Версия Minecraft: 1.21.11
Загрузчик: Fabric
Java: 21+
Размер файла: ~50 KB
